Documentation of our Inquizable API

Documentation:
get /users --> gets all the users
get /users/count --> gets the amount of users
get /users/answered --> gets the total amount of answered questions by users
get /users/correct --> gets the total amount of correctly answered questions by users
get /users/login --> login, requires email and pw
post /users/register --> register, requires nickname, email, pw
get /users/thisuser --> returns all the information of logged in user, requires authToken
get /users/{id} --> gets only the user with that ID
get /questions/all --> gets all the questions
get /questions --> gets 10 random, approved questions
get /questions/count --> gets the amount of questions
get /questions/{id} --> gets only the question with that ID
post /questions --> adds a question, needs type, category, difficulty, question, correct_answer, incorrect_answers, approved and Users_id
get /questions/submitted/{users_id} --> gets the amount of submitted questions that have been approved only by that user
get /users/points --> gets the nickname and points of all users, ordered by points DESC
get /users/contributions --> gets the nickname amounts of approved submitted questions from all users, ordered by amount DESC
get /users/percentage --> gets the nickname and % of correctly answered questions of all users, if atleast 30 questions have been answered, rounded to 1 decimal place and ordered by % DESC
